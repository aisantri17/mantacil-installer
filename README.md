# MantaCil Installer

Kumpulan script otomatis (auto-installer) buatan MantaCil Team. Script ini dibikin khusus buat kamu yang gak mau pusing ngetik command panjang-panjang pas mau install server. Tinggal copas satu baris, pilih menu, beres!

## Apa yang bisa diinstal?
1. **MantaCil Panel**: Panel server 3D Glassmorphism full Bahasa Indonesia + fitur Dewa Anti-Intip.
2. **MantaCil Wings**: Mesin belakang layarnya.
3. **MantaCil Security Toolkit**: Robot pintar (*cronjob*) penjaga server. Fitur ini bakal otomatis nge-banned/membekukan server pelanggan yang ketahuan make VPS buat DDoS, Flooder, atau nyedot CPU sampe lebih dari 95%.

## Cara Pakai
Pastiin VPS kamu masih fresh (baru diinstall OS) dan **wajib pakai Ubuntu 22.04 atau 24.04**. Langsung aja jalankan perintah saktinya:

```bash
bash <(curl -s https://raw.githubusercontent.com/aisantri17/mantacil-installer/main/install.sh)
```

Nanti bakal muncul menu interaktif. Kamu tinggal masukin pilihan mau install Panel doang, Wings doang, atau sikat dua-duanya sekalian.

## Disclaimer
Gunakan di server fresh biar gak ada bentrok sama aplikasi atau web server lain yang udah pernah kamu install sebelumnya.
